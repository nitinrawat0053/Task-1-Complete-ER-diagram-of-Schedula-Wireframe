## Task 2 – NestJS Boilerplate + Hello World API

## Overview
This task includes a minimal NestJS boilerplate setup with a simple GET endpoint that returns a JSON response.


## Setup

1. Navigate to the backend folder:

```bash
cd task-2-nestjs-boilerplate
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm run start
```
## API Endpoints

- **GET `/hello`**
  - **Response**:

```json
{ "message": "Hello World" }
```
